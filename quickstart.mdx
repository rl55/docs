## Get started in three steps

Get your site work with IduScore using REST APIs.

### Step 1: Request your API Key and a Partner Id

<AccordionGroup>
  <Accordion icon="copy" title="Request Auth Keys">
    During the onboarding process with IduScore, you should have received authentication keys - an API Key and a Partner Id.
    
    In case you did not receive these keys, get in touch with the support team at support@iduscore.com
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Test connectivity">
    1. Open the terminal window 
    2. Execute command: `curl https://api.iduscore.com/health` to test the connectivity and also the health of iduscore api server!
    
    <Tip>You should get a response like this: `{"status":"ok","timestamp":"2026-04-23T21:04:23.433Z"}`</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Deploy your APIs

<AccordionGroup>
  <Accordion icon="github" title="Setup your app">
    Use the provided [API documentation](/api-reference/submission).
    
    Our GitHub app automatically deploys your changes to your docs site, so you don't need to manage deployments yourself.
</Accordion>
<Accordion icon="palette" title="Update your site name and colors">
    For a first change, let's update the name and colors of your docs site.

    1. Open `docs.json` in your editor.
    2. Change the `"name"` field to your project name.
    3. Update the `"colors"` to match your brand.
    4. Save and see your changes instantly at `http://localhost:3000`.

    <Tip>Try changing the primary color to see an immediate difference!</Tip>
  </Accordion>
</AccordionGroup>

### Step 3: Go live

<Accordion icon="rocket" title="Publish your docs">
  1. Commit and push your changes.
  2. Your docs will update and be live in moments!
</Accordion>
